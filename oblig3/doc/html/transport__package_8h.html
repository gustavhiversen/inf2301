transport_package_t* transport_pkg_copy ( transport_package_t *  src)

Creates a complete copy of the specified transport package.

Parameters
src
Returns
A reference to a newly created transport package that is an exact copy of src , or NULL if an error occurred.

The returned copy of src is placed into one continuous memeory buffer containing both the transport package structure as well as a copy of the payload data contained within src . This allows all copied data to be deallocated as a single operation.

The returned transport package is expected to be deallocated using the transport_pkg_destroy function.

Exceptions
NULLsrc is NULL.
-or-
There is not enough memory available on the system to allocate a new transport package. errno is set to an appropiate error value.

Frees all memory occupeid by the specified transport package.

Parameters

If tp_pkg is NULL this function has no effect.

If the specified transport package is a copied transport package that was obtained through a call to transport_pkg_copy, this function will also automatically free the copy of the payload data.
