Field Documentation

◆ dataFirst

application_data_node_t* dataFirst

Indicates the current start of the data node queue for this application layer.

The value of this member will be a pointer to a application data node or NULL if the queue is empty.

The application layer data node queue stores the nodes of data that are expected to arrive from the underlying transport layer.

◆ dataGenerateCount

int dataGenerateCount

Indicates how many times this application layer has generated new data.

This value does not indicate how many bytes actually were generated.

◆ dataGenerateTimer

tick_timer_t dataGenerateTimer

The timer that belongs to this application layer.

When this timer times out, the application layer will generate new data and attempt to transmit it by sending it downwards in the OSI-stack.

◆ dataLast

application_data_node_t* dataLast

Indicates the current tail of the data node queue for this application layer.

The value of this member will be a pointer to a application data node or NULL if the queue is empty.

The application layer data node queue stores the nodes of data that are expected to arrive from the underlying transport layer.

The value of this member must not be NULL.
