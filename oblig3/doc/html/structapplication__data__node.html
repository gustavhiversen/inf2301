Field Documentation

◆ data

char data[MAX_APP_DATA_SIZE]

A buffer of static size MAX_APP_DATA_SIZE that contains the data stored in this node.

Only values with indicies from idx up to but excluding size can be meaningfully accessed within this buffer.

◆ idx

size_t idx

The index into the data member that indicates the first next byte of data that can be consumed in this node.

The value of the idx member must not exceed the value of the size member.

◆ next

application_data_node_t* next

A pointer to the data node that follows after this node or NULL if this is the final node.

◆ size

size_t size

The number of valid bytes of data that is stored in the data member of this node.

The value of the size member must not exceed MAX_APP_DATA_SIZE.

If the value of this member is 0U (zero) the node does not contain any meaningful data.
