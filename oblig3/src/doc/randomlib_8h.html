§ rand_normal_boxmuller()

double rand_normal_boxmuller ( )

Generates a normally distributed random value value centered around 0.0 (zero).

Returns
A double floating point value.

The default random number generator included in the C-runtime library generates uniformly distributed random values. The Box-Muller transformation algorthim is used to transform these uniformly distributed values to a normal distribution.

This function alternates between calling the Cosinus and the Sinus variant of the Box-Muller Transformation.

This function makes use of the rand function in the C-runtime library. srand may be used to seed the random number generator.

This function generates the deafult normal distribution. I.e. a normal distribution with a mean value of 0 (zero) and a standard deviation of 1.0.
