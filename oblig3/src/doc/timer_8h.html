§ tick_timer_callback_t

typedef void(* tick_timer_callback_t) (void *ctx)

The type representing a callback function that is invoked when a timer object reaches its targeted number of ticks.

Parameters
ctxA generic reference to an object representing the context of the expired timer.

The value of this parameter will be equal to the value of the ctx argument passed on to the original call to the timer_set function.

It is up to the implementation of the function whose signature matches the tick_timer_callback_t to provide meaning to the ctx parameter. Explicit casts to a more specific reference type will be necessary in most cases.

If the value of ctx is to be used in the callback, the caller of the orignal call to the timer_set function will need to ensure that the memory location referred to by ctx is still valid when the callback function is invoked.

Often it will be necessary that the callback function will perform cleanup actions for the ctx parameter value.

§ tick_timer_t

typedef struct tick_timer tick_timer_t

Type representing a tick-timer object.

A timer object has expired when its tick_count value has become equal to or greater than its target_ticks value.

void timer_set ( tick_timer_t *  timer,

Sets the specified timer object and starts its countdown towards expiration.

Parameters
timerA reference to the timer object to set.
ticksThe number of ticks until the timer object should expire.
callbackA reference to the callback function to invoke when the timer object expires.
ctxA generic reference to pass on to the callback function when the timer object expires.

If timer is NULL, this function has no effect.

ticks should be a positive value. Specifying a value of 0 (zero) for the ticks parameter will cause the timer object to expire immediately.

The meaning of the ctx parameter value is definied by the specific implementational details of the specified callback function.

If the caller specifies NULL as a value for callback , the timer objects will still expire after ticks ticks, but no callback function will be invoked.

This function overwrites all values of the specified timer object. The tick_timer_t::tick_count is reset to 0, and the other fields are set to the their respective parameter values passed as arguments to this function.
