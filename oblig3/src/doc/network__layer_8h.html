§ network_layer_create()

network_layer_t* network_layer_create ( osi_stack_t *  osi_stack)

Creates a new network layer instance for the specified OSI-stack.

Parameters
osi_stackA reference to the OSI stack for which the layer will be created.
Returns
A reference to the created network layer instance. It must be deallocated with a call to network_layer_destroy when it is no longer needed.

malloc is used to allocate memory on the system.

This function is implemented by the precode. Students are neither required to implement nor to understand the details of this function.

Exceptions
NULLosi_stack is NULL.
NULLThere is not enough memory available on the system to allocate a new network layer. errno is set to an appropiate error value.
