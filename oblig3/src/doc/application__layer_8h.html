application_layer_t* application_layer_create ( osi_stack_t *  osi_stack)

Creates a new application layer instance for the specified OSI-stack.

Parameters
osi_stackA pointer to the OSI-stack for which the layer will be created.
Returns
A pointer to the created application layer. It must be deallocated with a call to application_layer_destroy when it is no longer needed.

malloc is used to allocate memory on the system.
This function is implemented by the precode. Students are neither required to implement nor to understand the details of this function.

Exceptions
NULLosi_stack is NULL.
NULLThere is not enough memory available on the system to allocate a new application layer. errno is set to an appropiate error value.

§ application_layer_onTpReceive()

void application_layer_onTpReceive ( application_layer_t *  app_layer,
void *  data,
size_t  size 
)

Called when the application layer receives data from the underlying transport layer.

Parameters
app_layerA pointer to the application layer that receives the data.
dataA pointer to the buffer that contains the data. It must be at least size bytes long.
sizeThe number of valid bytes contained in the buffer pointed to by data .

If either app_layer or data is NULL, this function has no effect.
If size is 0 (zero), it signals the application layer that the last byte has been received and that no more communication is possible. This will cause the application layer to tear down the OSI-stack.
This function is implemented by the precode. Students are neither required to implement nor to understand the details of this function.
